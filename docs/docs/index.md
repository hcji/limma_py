# limma-py

一个基础的Python版limma实现，用于基因表达数据的差异表达分析。

## 特性

- ✅ **完整功能实现** - 提供与R版limma相同的核心功能
- ✅ **线性模型拟合** - 支持复杂实验设计矩阵
- ✅ **经验贝叶斯调节** - 提高小样本数据的统计功效
- ✅ **多组比较** - 支持复杂的对比分析
- ✅ **Pythonic API** - 为Python用户设计的友好接口
- ✅ **高性能计算** - 基于NumPy和SciPy的优化实现
